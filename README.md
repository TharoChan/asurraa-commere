
# E-Commerce Website

## Getting Started

1. Install dependencies:

```bash
pnpm install
```

2. Run the development server:

```bash
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **Redux**
- **Framer Motion**
- **ShadCN UI**
